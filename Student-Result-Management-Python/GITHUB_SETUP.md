# 🚀 GitHub Upload Guide

Follow these steps to upload BOTH projects to GitHub.

---

## For Each Project, Create a Separate Repository:

| Project | Repo Name |
|---|---|
| Library Book Tracker | `Library-Book-Tracker-C` |
| Student Result Management | `Student-Result-Management-Python` |

---

## STEP 1 — Create Repository on GitHub
1. Go to https://github.com → Click **"+"** → **"New repository"**
2. Set name, make it **Public**, do NOT add README (we have one)
3. Click **"Create repository"**

---

## STEP 2 — Upload from Your Computer

Open Terminal inside the project folder:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/REPO-NAME.git
git push -u origin main
```

---

## STEP 3 — Add to Resume

```
github.com/YOUR-USERNAME/Library-Book-Tracker-C
github.com/YOUR-USERNAME/Student-Result-Management-Python
```
